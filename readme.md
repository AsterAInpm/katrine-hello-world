# Katrine "Hello world" app

### step 0. Clone the app
    git clone https://github.com/AsterAInpm/katrine-hello-world

### Step 1. Install the demo app
    npm i 

### Step 2. Build the app
    npm run build

### Step 3. Run the app
    node ./

### Step 4. Type at browser address  

    http://localhost:2100
